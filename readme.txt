Tên đề tài: Thực hiện một hệ thống giảng dạy trực tuyến tương tự như Google Classroom

Thành viên:
+ Huỳnh Anh Tài - MSSV: 51702171
+ Trần Phương Ngọc Anh - MSSV: 51702061

Setup môi trường:
Bước 1: Cài đặt và cấu hình phần mềm Xampp: https://www.apachefriends.org/index.html
Bước 2: Giải nén thư mục 51702171-HuynhAnhTai-51702061-TranPhuongNgocAnh.zip và đặt trong thư mục htdocs của Xampp.
Bước 3: Run tệp xampp-control.exe và bấm Start Apache và MySQL để khởi động web server.
Bước 4: Truy cập đường dẫn http://localhost/phpmyadmin để import file database database.sql
Bước 5: Truy cập đường dẫn http://localhost và bắt đầu sử dụng.

Bảng phân công công việc:
+ Huỳnh Anh Tài: thiết kế database, làm logic web
+ Trần Phương Ngọc Anh: thiết kế UI, làm UI web
